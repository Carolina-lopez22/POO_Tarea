#Documentación
En estos ejercicios realizados pude comprender mejor el manejo de atributos, metodos, validaciónes y diferentes estructuras en java 
se pudo apreciar como el trabajar en diferentes clases da un mejor dominio a la hora de correguir errores, ya que sabia exactamente que 
ejercicio era el que estaba generando error, trabaje con atributos private para representar las caracteristicas que necesitaba cada objeto
como por ejermplo nombre, precio, autor, etc.

Para poder manipular estos datos implemente metodos, que devolvieran valores como los metodos que use para el ejercicio de calculadora, tambien 
estos metodos me ayudaron a realizar acciones que requerian varios ejercicios, como aplicar descuentos o validar condiciones como lo fue en el
caso del los libros. En resumen utilice metodos return para devolver los resultados requeridos y metodos void para modificar directamente los
atributos de los objetos.

Para cumplir con los requisitos de validaciones que se me solicito en casi todos los ejercicios use la validacion if para evitar la divisiones entre
cero por ejemplo otra a destacar fue para evitar las operaciones con numero negativo en el ejercicio de cuenta. Un tipo de dato que utilice fue boolean 
para evaluar las condiciones logicas, como determinar si un libro era largo y para verificar si las fechas eran correctas.

Asimismo, implemente el uso de constructores para inicializar los objetos con valores específicos al momento de crearlos.

Finalmente, se aplicó el principio de encapsulamiento, utilizando atributos private y métodos como getters y setters para controlar el acceso a los datos.
